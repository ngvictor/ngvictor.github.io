Type.

Inspire, Discover, Experiment

Type is a mobile application dedicated to the end to end workflow for designers and typography enthusiasts to find, explore, inspire, and experiment with digital typography.


Type offers 5 features below:

Scan

Designed for you to find the exact font you are looking for. Unrestricted by location, users can simply take a photo or upload one from their album to scan the font. Type then looks for the font or closest looking font.

This is intended to be the core feature of the application, leveraging the mobility and camera of a smartphone.

Scanning of a poster using the camera. The Scan icon is located in the center of the navigation bar.

Screenshot of the identified font. Users can save it to their folders and see related articles/fonts-in-use about that font.

Home Feed

The home feed is an area for inspiration and education; it is also the landing page of the applicaiton. Users can search for articles and applications of a particular typeface.

Font in use exhibits posters and digital media with creative or appealing applications of type. Articles are could be about anything related typography and not limited to education and font features.

Scrolling of Font in use and Article feeds. Font in use contains only images unless selected.

Exploring inside Font in Use page. In the selected image, the font used is displayed as well as other related images. Users can save this content to a specific folder.

Exploring inside of article. The designed is based on reading platforms like Medium. Users are also able to save this content to a specific folder.

Catalog

A secondary area for inspiration and education; it works as a repository of all fonts housed within Type. Type users can browse font faces and test font pairings for their next project.

Scrolling of the catalog page with an expandable filter at the top.

Exploring inside a typeface. Sections include all of the type's different styles, a Type bio, and Font pairings.

Switching between font pairings where users can experiment pairings of the selected font.

Expanded Catalog Filter

Prototype

Feature for users to test fonts and font pairings on different colors and backgrounds. Templates are designed to save users time. Saved fonts are based on saved content.

Buttons include: Textbox, Image, Template, Text color, Background color.

Enlarging the font size in the detailed view. Different text options are viewed one at a time due to the limitations of the screen size.

Dragging of template items into the prototype canvas.

Designed to streamline font comparisons, users can simply drag out sample text from the template panel with prebuilt sample text.
